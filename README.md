# my_world_model

自身の生命に対する認識のメモ書き

## 見ている人へ

弱ってスピリチュアルな世界に行こうとしているわけではありません。基本、自分は思春期あたりからいつもこういうことを考えているのが好きで、ただそれだけだと俗世と離れてしまうので、仕方なしに仕事したり遊んだりしてるという感じです。ここでは単に素の自分を出しているだけです。大丈夫です。

このまま仕方のない生活だけをしていると子供のときの自分に申し訳が立たないので、死ぬまでにちゃんと文章に落とそうとしてたのですが、やっとやる気になったのが今日ということです。

## この文章のアプローチ

研究や学問で作法とされる文献調査や引用、議論で深めていくのではなく、自身の頭の中のモデルをひたすら記述していく。もちろんそのなかで必要なことは引用するが、おそらく哲学的なものではなく、医学や制御、ロボット工学、認知科学、心理学の論文が中心となる。

# 最初の原稿（そのうちTeXに移行）

生命は増えるから生命。おそらく宗教ではなくて最適化問題のほうが説明しやすい。

高度な神経回路を持つ動物は生活圏のモデルを頭の中に持っている。そのモデルが正しいかどうかは、人間以外ではその個体が生き残ったり殖えたりすることで担保される側面が強い。人間の場合はモデルについて自身で他者に伝えることができるので、もう少し複雑。

植物の場合はモデルを持ってモデルから行動を決めると言うよりは受動的な側面が強い。ただ、制御系ととらえたときに動物と植物の境界はそんなにないのではないか。（ブライテンベルグビーグルの話など）
